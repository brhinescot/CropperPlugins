Cropper TFS Work Item Plugin

Cature images using Cropper directly to a new or an existing work item attchments field.

 Requirements

Microsoft Team Explorer 2008.

Installation
Copy the files to cropper plugins directory: C:\Program Files\Fusion8Design\Cropper\plugins.
Copy the following files from the VisualStudio PrivateAssemblies folder (for example, C:\Program Files\Microsoft Visual Studio 9.0\Common7\IDE\PrivateAssemblies\Microsoft.TeamFoundation.dll) the plugins directory:
  1. Microsoft.TeamFoundation.WorkItemTracking.Controls.dll
Usage
  • Open Cropper and select TFS Work Item from the Output menu.
  • The first time you snap a screen shot with TFS selected as the output, the TFS options dialog will appear.
  • When capturing and image using Cropper a window with the default options will be opened. After pressing OK the work item editor window will be opened with the captured image as an attachment.
  • To edit TFS options at any other time, select the "Options" menu choice from the Cropper popup menu. Select the Plugins tab. In the combo box, select the TFS plugin.
