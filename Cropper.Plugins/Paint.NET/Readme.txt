Readme for Cropper.Plugins/Paint.NET


This is a plugin for Cropper, it sends the image to Paint.NET.

To use it, just drop the DLL into the plugins directory, and restart
cropper.  When you capture a screen image, it will be saved, and then
Paint.NET will be opened with that saved image.
