Cropper TFS Work Item Plugin

Cature images using Cropper directly to a new or an existing work item attchments field.

 Requirements

Microsoft Team Explorer.

Installation
Copy the files to cropper plugins directory: C:\Program Files\Fusion8Design\Cropper\plugins.
Copy the following files from the VisualStudio PrivateAssemblies folder to the plugins directory:
  1. Microsoft.TeamFoundation.WorkItemTracking.Controls.dll
Usage
  • Open Cropper and select TFS Work Item from the Output menu.
  • For the first time the options dialog will appear.
  • When capturing and image using Cropper a window with the default options will be opened. After pressing OK the work item editor window will be opened with the captured image as an attachment.
  • To open the options dialog when specified: "Do not show this dialog again" hold the Shift key while selecting TFS Work Item from the Output menu.
